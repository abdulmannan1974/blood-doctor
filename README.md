# Blood Doctor

**Interactive Haematology Learning & Clinical Tools**

A mobile-first web application providing educational resources, clinical tools, and interactive guides for haematology professionals, trainees, and educators.

## About

Blood Doctor is a collection of haematology projects built as progressive web apps, designed for mobile use via Claude Code. Each project focuses on a specific area of haematology, delivering evidence-based educational content in an accessible, interactive format.

## Projects

Projects will be organized in individual directories:

```
blood-doctor/
├── README.md
├── projects/
│   ├── [project-name]/
│   │   ├── index.html
│   │   └── ...
│   └── ...
└── shared/
    ├── css/
    └── js/
```

## Tech Stack

- HTML5 / CSS3 / JavaScript (vanilla)
- Mobile-first responsive design
- Single-file apps for easy deployment
- No build tools required
